## zh-CN

使用 `setMessages` 方法可批量设置或更新对话消息列表，常用于加载远端历史消息或初始化对话状态。该方法会替换当前所有消息，适用于首次加载历史记录或需要重置对话内容的场景。

## en-US

The `setMessages` method allows you to batch set or update the conversation message list, commonly used for loading historical messages from remote sources or initializing conversation state. This method replaces all current messages and is suitable for initial historical data loading or conversation content reset scenarios.
