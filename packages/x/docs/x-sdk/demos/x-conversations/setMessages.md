## zh-CN

使用 `setMessage` 方法可为指定会话设置消息内容，支持更新现有会话的历史消息记录。该方法常与 `useXConversations` hook 配合使用，实现会话数据的动态管理和状态同步，适用于会话列表的消息更新和会话内容的初始化场景。

## en-US

The `setMessage` method allows you to set message content for a specific conversation, supporting updates to existing conversation history records. This method is commonly used in conjunction with the `useXConversations` hook to achieve dynamic conversation data management and state synchronization, suitable for conversation list message updates and conversation content initialization scenarios.
